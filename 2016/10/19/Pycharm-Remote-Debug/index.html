发表于 2016-10-19
  • 在远程客户端[linux]安装pycharm-debug.egg 首先，下载安装easy_install的安装脚本

pycharm-debug.egg文件在Pycharm安装目录下的debug-eggs文件夹中

类似地，在服务端[Windows]中安装pycharm-debug.egg

  • 在Pycharm中配置Remote Debug Edit Configuration->+>Python Remote remote_debug

    Local host name 填写Windows服务端IP        
Port 填写任意未被占用端口
